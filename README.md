## Project Description
[Podo kayıp. Ankara vakası 5 tane Jotform formu (checkins,messages,sightings,Personal Notes, Anonymous Tips.) üzerine kurulu bir araştırma dashboradu.Her submissionı fetchleyip bunları tek bir case viewa çeviriyoruz. Bir sıralı şüpheli listesi, vaka özeti çizelgesi, Podo'nun aldığı yol, Ankaradaki ihbar edilen yerlerin olduğu bir mini map ve filtrelenebilir event feedden oluşuyor. Herhangi bir componenta tıklamak (suçlu, ihbar, harita ibresi vb.) daha detaylı açıklamalar gösteriyor. ]

## Getting Started
[Before you start, make sure you have:

Node.js 18 or higher ,
npm,
Git.

1.Enter Project Folder
cd 2026-frontend-challenge-ankara

2. Install dependecies

3. Start Development Server
npm run dev

4. Open the app in your browser
Go to http://localhost:5173]
